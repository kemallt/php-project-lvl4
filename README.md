## Requirements
* PHP ^8.0
* Composer
* SQLite or PostgresQL


### Setup
```shell
$ make setup
```
### Run
```shell
$ make start
```
### Test
```shell
$ make test
```
### Lint
```shell
$ make lint
```
### Deploy
```shell
$ make deploy
```
